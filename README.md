# 🎮 Adem Uçar — Kişisel Portfolyo & Web Uygulamaları

Kendi geliştirdiğim **interaktif web oyunları** ve günlük hayatı kolaylaştıran **hesaplama araçlarını** bir araya getirdiğim kişisel portfolyo sitem.

🔗 **Canlı site:** [ademucarr.netlify.app](https://ademucarr.netlify.app/)

---

## 🕹️ Oyunlar

| Oyun | Açıklama |
|------|----------|
| **Neon Snake** | Klasik yılan oyununun neon temalı modern yorumu |
| **Neon Bird** | Boru engellerini aşarak yüksek skora ulaşılan minimalist kuş oyunu |
| **Neon Tetris** | Neon çizgilerle tasarlanmış klasik blok birleştirme oyunu |
| **Neon Wordle** | Türkçe kelime havuzuna sahip, deneme hakkı sınırlı kelime bulma oyunu |
| **Neon Flagle** | Bayrak parçalarını açarak ülke tahmin edilen coğrafya oyunu |
| **Bir Kelime Bir İşlem** | Harflerden en uzun kelimeyi veya sayılarla hedefe ulaşmayı amaçlayan oyun |

## 🛠️ Araçlar

- **Çalışma Programı Oluştur** — kişiye özel çalışma planı hazırlama aracı
- **Gano Hesapla** — ağırlıklı not ortalaması (GANO) hesaplama aracı
- **Dönem Sayacı** — dönem sonuna / önemli tarihlere geri sayım
